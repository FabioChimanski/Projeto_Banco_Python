##Sistema Bancário em Python com SQLite##
Este é um projeto de estudo focado em Programação Orientada a Objetos e persistência de dados utilizando o banco de dados SQLite. O objetivo foi criar um simulador de caixa eletrônico funcional, aplicando conceitos reais de arquitetura de software.

##Funcionalidades##
- **Cadastro de Usuários**: Criação de contas com armazenamento persistente.
- **Autenticação**: Sistema de login seguro consultando registros no banco de dados.
- **Operações Financeiras**:
  - Consulta de saldo em tempo real.
  - Saque.
  - Depósito.
- **Persistência de Dados**: Todas as alterações são salvas automaticamente no banco de dados SQLite.

##Tecnologias Utilizadas##
- **Python**
- **SQLite3**
- **Arquitetura Modular**

##Conceitos Aplicados##
- Programação Orientada a Objetos (Classes, Atributos, Métodos, `__init__`).
- Comandos SQL (CRUD: Create, Read, Update).
- Tratamento de exceções com `try/except`.
- Manipulação de arquivos e caminhos em Python.
